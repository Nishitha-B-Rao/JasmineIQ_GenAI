# JasmineIQ Frontend

This is the Next.js 15 web application for the JasmineIQ Decision Intelligence Platform.

## Features
- **Live Dashboard**: View scraped market rates, XGBoost predictions, and localized weather metrics.
- **AI Recommendations**: Explanations generated by Google Gemini Flash.
- **Revenue Calculator**: Interactive calculator for farmers to estimate harvest value.
- **Analytics Charts**: Historical data rendering using Recharts.

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Environment Variables
Create a `.env.local` file to point to your FastAPI backend:
```
NEXT_PUBLIC_API_URL=http://localhost:8000/api
```
